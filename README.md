# Introduction 
API for Ecom Pilot

#Prerequisite:
dotnet(10.x): https://dotnet.microsoft.com/en-us/download/dotnet/10.0
entity framework: `dotnet tool install --global dotnet-ef`
postgres(latest): https://www.enterprisedb.com/downloads/postgres-postgresql-downloads

# How to install?
1.  Pull the project from github,
2.  Open command prompt, and run 
    `dotnet new --install {root path of the project}`
3.  The template should get successfully installed.

# How to create a new project using this template?(Visual Studio)
1.  After the installation process is complete, open Visual Studio
2.  Create a new project
3.  Search and select *EcomAPI*
4.  Follow the steps
